# Green-light days — the best days to start, buy, launch

The opposite of the gates: festival/auspicious days when a new start needs **no further muhurta** (still dodge the day's Rahu Kaal). Confirm exact dates on a panchang each year (they move).

| Day | ~When | Best for |
|---|---|---|
| **Akshaya Tritiya** | Apr/May (Vaishakha Shukla 3) | *The* day for new undertakings, gold/property/tool buying, launches — "never-diminishing" |
| **Dhanteras** | Oct/Nov (pre-Diwali) | Buying metal, machinery, tools, vehicles; wealth |
| **Diwali / Lakshmi Puja** | Oct/Nov | New account books (Chopda Pujan), wealth, fresh financial start |
| **Gudi Padwa / Ugadi** | Mar/Apr (Chaitra Shukla 1) | New-year starts, launches, openings |
| **Vijayadashami / Dussehra** | Sep/Oct | Starting anything new; **Ayudha/Shastra Puja** (tools); "victory" undertakings |
| **Navratri (any day)** | Mar/Apr & Sep/Oct | Auspicious window for new starts (Shakti) |
| **Vasant Panchami** | Jan/Feb | Learning, design, knowledge work (Saraswati) |
| **Makar Sankranti** | ~14 Jan | Ends Kharmas — auspicious starts resume |
| **Guru / Ravi Pushya yoga** | monthly | Pushya nakshatra on a **Thursday (Guru)** or **Sunday (Ravi)** — a recurring "mini-Akshaya-Tritiya," gold for **buying/investing** |

**Use:** when the user wants "a good day soon," offer the **next green-light day** and run `panchang.py` to avoid Rahu Kaal. **Guru-Pushya recurs monthly** and is excellent for purchases — flag it whenever it's near (your raw-material orders, machinery, vehicles).
