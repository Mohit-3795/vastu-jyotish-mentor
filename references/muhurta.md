# Muhurta, choosing an auspicious time

**Order of checks, never skip Step 0.** Run `scripts/panchang.py <date>` first; it does Steps 0-2 for you.

### Step 0, PERIOD GATE (do this FIRST)
See `calendar-gates.md`. If **Adhik Maas, Kharmas, an eclipse, or Pitru Paksha** is active, the period is blocked, **the answer is simply "wait until {it ends}."** Don't score individual days inside a blocked period. *(2026 Adhik Maas = 17 May-15 Jun.)*

### Step 1, DAY (only if the gate is clear)
- **Waxing moon (Shukla paksha)** for anything you want to grow.
- **Good weekday for the activity** (table below).
- **Good nakshatra**; avoid **Rikta tithis (4th, 9th, 14th)** and Amavasya.

### Step 2, TIME
- Avoid **Rahu Kaal** (the script prints the exact window for the day/place).
- For a personal "first switch-on," it's a bonus if the Moon isn't in the 4th/8th/12th sign from the person's natal Moon (**Chandrabala**), confirm on a panchang if it matters.

## Weekday by activity
| Day | Planet | Best for | Avoid for |
|---|---|---|---|
| **Thursday** | Jupiter | New businesses, finance, expansion, signings, registration, **most auspicious; principal's lagna lord** |, |
| **Wednesday** | Mercury | Trade, contracts, accounts, communication, deals |, |
| **Friday** | Venus | Launches, partnerships, branding, openings |, |
| **Sunday** | Sun | Leadership, authority/govt dealings, **company number 1** |, |
| **Monday** | Moon | Public-facing, soft starts | Hard commitments |
| **Tuesday** | Mars | Machinery install, property, bold moves | Loans, delicate finance |
| **Saturday** | Saturn | Labour, iron, long-term/foundational work | New auspicious *starts* |

## Nakshatra (quick)
- **Strong for starts/business:** Pushya (best), Hasta, Chitra, Swati, Anuradha, Rohini, Ashwini, the Uttara set, Shravana, Revati.
- **Weak for starts:** Bharani, Krittika, Ashlesha, Jyeshtha, Mula (Magha mixed).

## Tailoring to the user (read `profile.md`)
- Favour the **principal's lagna-lord weekday** + the **company-number day** for big moves; gate clear + **waxing moon + Pushya/Hasta**, outside Rahu Kaal = the sweet spot for launches/registrations.
- **Tuesday** for installing machinery; **Wednesday** for contracts/accounts; **Thursday** for launches/finance.
- Prefer dates reducing to the **principal's / company's lucky numbers** (profile.md).
- **Buying** is a Mercury act, don't purchase under **Mercury retrograde** (the script flags it).
